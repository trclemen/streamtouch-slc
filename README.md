# StreamTouch Local Cloud (SLC)

Local media server that enables hardware preset saving 
for SoundTouch speakers via StreamTouch.

## Quick Start

1. Copy `docker-compose.yml` and edit the environment 
   variables to match your network
2. Run `docker compose up -d --build`
3. Check `http://[your-slc-ip]:8300/api/health`
